HJImagesToGIF
=============

Convert image array to animated GIF

### Save to path
```
[HJImagesToGIF saveGIFFromImages:images toPath:path WithCallbackBlock:nil];

```
Save to photo album
```
[HJImagesToGIF saveGIFToPhotoAlbumFromImages:images WithCallbackBlock:nil];

```
